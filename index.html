modhex or else!

Download modhex.js.

Frustrated with the current state of yubikey's keyboard layout compatibility, I decided to write a little program that iterates through over 400 keyboard layouts to find which key would be produced in that layout by the yubikey. It turns out the compatible keymaps (almost all of them) boil down to 93 unique sequences of 16 characters, or 93 different ways to convert the yubikey's output to modhex or just normal hexadecimal. By efficiently comparing the yubikey's output to these keymaps, we can almost always find just one or two possible modhex interpretations of the one time password.

Of course, this ought to happen on the server. But it's probably easier for most to drop in a little javascript than extend the server to check each possible interpretation of the entered passwords.

Go ahead. Change your keymap. Russian? Uzbek? French dvorak? Japanese kana?

They will all work. If you find a layout that doesn't work or if you enjoy this program please let me know at dholth@gmail.com.
